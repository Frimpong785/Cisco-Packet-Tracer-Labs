# Trunking, Allowed VLANs and Native VLAN

## Objective

The objective of this lab is to configure IEEE 802.1Q trunking between Cisco switches and manage VLAN traffic using allowed VLAN lists, and native VLAN configuration.

## Devices Used

- 2 Cisco 2960 Switches
- 4 PCs

## Configuration Tasks

- Configure trunk links
- Configure the native VLAN
- Configure the allowed VLAN list
- Verify trunk operation
- Verify VLAN propagation

## Verification Commands

```bash
show interfaces trunk
show vlan brief
show interfaces switchport
show running-config
```

## Skills Practiced

- 802.1Q Trunking
- Native VLAN Configuration
- Allowed VLAN Configuration
- VLAN Verification
