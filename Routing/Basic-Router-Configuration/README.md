# Basic Router configuration

## Objective

The objective of this lab is to configure a Cisco Layer 3 router with basic security and management settings.

## Topology

- 1 Cisco 2911 Router
- 1 PC
- console cable

## Configurations performed
- Changed the switch hostname
- Disabled DNS lookup
- Configured the enable secret passwords
- Encrypted plaintext passwords
- Configured the console password
- Configured VTY (SSH) password
- Configured a login banner 
- Prevent Brute-force attack
- Save the running configuration

## Verification commands

```bash
show running-config
show startup-config
show ip interface brief
```
