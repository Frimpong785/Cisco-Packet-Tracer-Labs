# Basic switch configuration

## Objective

The objective of this lab is to configure a Cisco Layer 2 switch with basic security and management settings.

## Topology

- 1 Cisco 2960 Switch
- 1 PC
- console cable

## Configurations performed
- Changed the switch hostname
- Disabled DNS lookup
- Configured the enable secret passwords
- Encrypted plaintext passwords
- Configured the console password
- Configured VTY (Telnet) password
- Configured a login banner 
- Management IP with a default gateway
- Save the running configuration

## Verification commands

```bash
show running-config
show startup-config
show ip interface brief
```
