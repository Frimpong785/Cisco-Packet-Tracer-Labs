# Notes

## Hostname

The hostname identifies the switch on the network and in the CLI.

## no ip domain-lookup

Disables DNS lookup to prevent delays when an incorrect command is entered.

## enale secret

Sets an encrypted password for privilaged EXEC mode.

## service password-encryption

Encrypts plaintext passwords stored in the configuration.

## Banner MOTD

Displays a warning message before users log in.

## Console password

Protect physical access through the consol port.

## VTY Password

Protect remote access.

## copy running-config startup-config

Saves the current configuration so it is retained after a reboot.
